
## Installation

1. Install [Python](https://www.python.org/)
2. Download Source Code
3. Ekstrak File
4. Buka code editor
5. Buka folder yang sudah diekstrak
6. jalankan perintah diterminal :
```bash
pip install bokeh
```
7. Setelah selesai install bokeh jalankan perintah :
```bash
python chart.py
```
